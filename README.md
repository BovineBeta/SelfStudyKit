# SelfStudyKit
# :warning:此项目的最初版本未完成
基于 [MDUI](https://github.com/zdhxiong/mdui) 的纯前端自习辅助工具集.

# Feature

- Material Design 设计, 界面简洁
- 基于HTML5, 跨平台

# Development
准备:
~~~bash
git clone https://github.com/BovineBeta/SelfStudyKit.git
cd SelfStudyKit
npm install
~~~
运行webpack dev server:
~~~
npm run start
~~~
构建:
~~~
npm run build
~~~
实时构建 (可搭配vscode扩展live server使用):
~~~
npm run build-watch
~~~
